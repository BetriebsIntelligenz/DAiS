- eintrag von neuen anforderungen (es öffnet sich ein pop up window zur eingabe einer aufforderung)
- folgende werte sollen minimum eingebbar sein : anforderungs zieldatum, anfrager, kosten, prio, bereich dropdown (Privat, Finanzen, Arbeit, Staat)

- die karten haben einen status (Offen, In Bearbeitung, Problem, Fertig), mit dem sie in die einzelenn bereiche des kanban board sortiert werden, der user kann den status ändern in der pop up ansicht, dannach wird die karte automatisch neu eingeordnet.
