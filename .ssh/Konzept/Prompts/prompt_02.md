Implementiere weitere programms in den bereich "mind". setzte die neuen programms genauso um wie die bestehenden.

# Visualisierungstraining
- es soll möglich sein im admin bereich neue fotos hochzuladen, die dann in diesem bereich gezeigt werden. in diesem bereich sollen bilder unterander anzeigt werden mit einer checkbox darüber zum anhacken

# Ziele
- es sollen ziele angezeigt werden und im admin bereich hinterlegt werden können. ziele sollen mit der methodik SMART formuliert werden.
- diese interface soll der user seine ziele ansehen, sie täglich abhacken können wenn er sie durchliest und soll sehen wie nah er seinem ziel schon ist und eine aktuelle selbsteinschätzung hinterlegen können.

# Brain Training
- hier soll der user kurze und bewerte brain übungen durchführen können. die übungen werden beschrieben und der user kann sie als durchgeführt markieren. dieser bereich soll ein trainings gym für das gehirn darstellen

# Higher Thinking
- hier sollen der bereich für lernen, philosophie und konzepte sein. der user soll sich in verschiedenen lernpfaden befinden die er einsehen kann und soll dabei kleine meilensteine abhacken können die er bereits gelernt hat um am ende den ganzen lernpfad durchlaufen zu haben.

# Emotion Training
- das soll ein hilfsbereich sein, wo man emotionen tracken kann und wo man übungen findet wie man mit emotionen umgeht
